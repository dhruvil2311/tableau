# Tableau Data Visualization Project

This repository contains interactive data visualizations and dashboards built using Tableau. The project focuses on transforming raw data which is used in sql for global covid dataset into impactful insights.
## Project Overview
The covid vaccinations and covid deaths dataset are divided into 4 excel file which are used for analysis

* **Dashboard File:** `Book1.twbx` (Tableau Packaged Workbook)

## Features & Visualizations Included
* **Interactive Dashboards:** Built-in filters and actions allowing users to drill down into specific data points.
* **Key Performance Indicators (KPIs):** High-level summaries tracking critical metrics at a glance.
* **Geographic Mapping:** Spatial analysis showcasing performance across different regions or territories.

## Tools & Technologies Used
* **Business Intelligence:** Tableau Desktop
* **Data Source:** SQL repository
